### Letöltés/használat

A projekt fontosabb adatai:
* Python 3.13.3
* pip 25.1.1
* Minden használt package megtalálható a `requirements.txt` nevű fájlban. (A `pip install -r 'requirements.txt'` parancs segítségével ezeket gyorsan és kényelmesen lehet letölteni.)
